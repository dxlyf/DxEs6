<static> request(options)

网络请求
Parameters:
Name Type Description
options object jQuery.Ajax（options）参数 可以参考jquery ajax
Properties
Name Type Argument Default Description
name string 接口配置名称，当为空null时，读取url
isCustomerError boolean <optional>
 false 默认系统处理错误信息，设为true自己处理
isWrapAjax boolean <optional>
 true 默认强制后台返回特定格式，设为false不要求
inShowLoading boolean <optional>
 true 默认请求后台会显示加载动画，设为false为不显示动画
Returns:
ajax promise 对象
Type
promise
